## 快捷操作一览

| 操作 | 方式 |
|------|------|
| 📁 目录树 | 左侧边栏浏览，点击加载文件 |
| 🔍 全文搜索 | `Ctrl + K` 打开搜索框 |
| ↔️ 切换侧边栏 | `Ctrl + B` |
| ✏️ 编辑模式 | 点击 Edit 按钮或 `Ctrl + S` 保存 |
| 🎨 切换主题 | 右下角按钮，16 种代码风格可选 |
| 📸 截图 | 截图按钮，一键保存为图片 |
| ⬅️ 关闭弹窗 | `Esc` 键关闭当前弹窗 |
