# 仓位管理模型

## 核心问题

> 每次交易应该押多大的注？

### Kelly 公式

$$f^* = \frac{bp - q}{b}$$

其中：
- $f^*$：最优仓位比例
- $b$：赔率（盈利/亏损比）
- $p$：胜率
- $q = 1-p$：败率

### 实际应用

| 场景 | Kelly 仓位 | 建议仓位 (1/3 Kelly) |
|------|-----------|-------------------|
| p=55%, b=2 | 32.5% | 10.8% |
| p=60%, b=1.5 | 33.3% | 11.1% |
| p=50%, b=1 | 0% | 0% |

## 风险预算

### 波动率目标
```
目标波动率 = 15%
组合波动率 = 25%
仓位比例 = 15% / 25% = 60%
杠杆 = 0.6x
```

### VaR 限制
```
1日 95% VaR 限制 = 2%
持有仓位 VaR ≤ 2%
基于历史模拟或参数法
```

## 条件仓位

基于市场状态的动态调整：

| 市场状态 | 基准仓位 | 调整系数 |
|----------|---------|----------|
| 强趋势 | 100% | 1.0x |
| 震荡 | 60% | 0.6x |
| 高波动 | 40% | 0.4x |
| 恐慌 | 20% | 0.2x |
