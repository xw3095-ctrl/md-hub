# 市场情绪指标

## 情绪测量的维度

### 1. 价格衍生指标
- 涨跌比 (Advance/Decline Ratio)
- 新高新低比 (52-Week High/Low)
- 相对强弱指标 (RSI)

### 2. 交易行为指标
- 成交量异常
- 换手率
- 融资融券余额变化

### 3. 期权衍生指标
- Put/Call Ratio
- 隐含波动率偏斜 (Skew)
- VIX 指数

## 前景理论 Lambda 指标

结合行为金融学的前景理论框架：

```python
def lambda_sentiment(r, rr, lambda_=2.25):
    """
    r: 收益率序列
    rr: 参考点 (参考收益率)
    lambda_: 损失厌恶系数
    """
    gains = r[r > rr]
    losses = r[r < rr]
    
    if len(gains) == 0 or len(losses) == 0:
        return 0.5
    
    gain_sentiment = (gains - rr).mean()
    loss_sentiment = lambda_ * (rr - losses).mean()
    
    return gain_sentiment / (gain_sentiment + loss_sentiment)
```
