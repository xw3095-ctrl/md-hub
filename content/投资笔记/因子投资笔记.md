# 因子投资笔记

## 五大经典因子

### 1. 市场因子 (Market)
- $\beta$：组合对市场的敏感度
- 长期为正，但波动较大

### 2. 价值因子 (Value)
- 低估值股票跑赢高估值
- 指标：PB、PE、PS
- A 股表现：早期显著，近年减弱

### 3. 规模因子 (Size)
- 小盘股跑赢大盘股
- A 股小盘效应明显
- 注意：壳价值影响

### 4. 动量因子 (Momentum)
- 过去 12 个月（除最近 1 个月）的收益率
- A 股动量较短（3-6 个月）
- 反转效应更强

### 5. 质量因子 (Quality)
- ROE、毛利率、杠杆率
- A 股质量因子近年表现突出

## 多因子组合

```python
def factor_portfolio(factors, weights):
    """构建多因子投资组合"""
    score = sum(w * f for w, f in zip(weights, factors))
    return score
```
